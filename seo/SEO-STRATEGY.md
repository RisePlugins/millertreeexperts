## On-Page SEO Specifications

### Homepage Title Tag
```
Miller Tree Experts | #1 Rated Tree Service in Berryville, VA | 5.0 ★
```

### Homepage Meta Description
```
Professional tree removal, trimming, stump grinding & storm damage cleanup in Berryville, Winchester & the Shenandoah Valley. 5.0 rated with 115+ reviews. Free estimates. Call (540) 313-1306.
```

### Schema Markup (Required on Every Page)
- **LocalBusiness** schema with full NAP, hours, service area
- **Service** schema for each service page
- **Review/AggregateRating** schema (5.0, 115 reviews)
- **FAQ** schema on service pages
- **BreadcrumbList** schema for navigation
- **Organization** schema with logo, social profiles

### Technical SEO Requirements
- Mobile-first responsive design
- Core Web Vitals: LCP < 2.5s, FID < 100ms, CLS < 0.1
- HTTPS (SSL certificate)
- XML sitemap auto-generation
- robots.txt with sitemap reference
- Canonical tags on all pages
- Open Graph and Twitter Card meta tags
- Image optimization (WebP format, lazy loading, alt text with keywords)
- Internal linking structure between service pages, location pages, and blog
