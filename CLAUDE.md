# Miller Tree Experts - Website Project

## Project Overview
Building a professional website for Miller Tree Experts LLC, a tree-care company based in Berryville, VA. The site will be a custom WordPress theme hosted on Hostinger. We are prototyping with plain HTML/CSS/JS first, then converting to a WP theme.
